# 群島之歌(Archipelago Song)

**🎮 線上遊玩:https://chenyiray0517-hub.github.io/archipelago-song/**(瀏覽器直接玩,建議電腦 + Chrome)

3D 海島探索 ARPG/開放世界。玩家駕船航行群島、收集「靈紋寶石」解鎖能力(類銀河惡魔城式探索循環),經驗值以道具「經驗結晶」形式存在背包、由玩家決定升級時機。

- 企劃書:`docs/群島之歌_遊戲企劃書.pdf`(v1.0,2026-06)
- 技術棧:**HTML5 + TypeScript + Three.js**(2026-06-12 定案,不用 Unity/UE5;Three.js 僅作渲染層,遊戲邏輯全自寫)
- 目標平台:瀏覽器(後續可包 Electron/Tauri 上 Steam)

## 開發

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # tsc 嚴格檢查 + vite build
npm run smoke    # Playwright 煙霧測試(需先啟動 dev server)
```

## 目前內容(原型期)

- 第三人稱角色控制:WASD 移動、跳躍、Shift 閃避(無敵幀 + 體力消耗)、右鍵環繞鏡頭、滾輪縮放
- 即時戰鬥:左鍵揮劍(扇形範圍判定)、敵人受擊閃紅 + 浮動血條
- 舉盾防禦(Q 按住):正面 ±75° 傷害減 85%,每擋耗 10 體力,移速下降,格擋火花 + 金屬聲
- 集氣迴旋斬(左鍵按住):劍身發光集氣 1 秒滿氣(脈動 + 提示音),放開後手臂前伸水平掃 360°,傷害 1x→2.5x + 大擊退;滿氣加放新月衝擊波(直線飛行 ~20u,路徑穿透命中)
- 敵人 AI:史萊姆 ×12 + 菁英怪 ×2(出生地新手怪 → 內陸丘陵漸強),巡邏/追擊/攻擊狀態機,死後 25 秒重生
- 經驗結晶系統(企劃書核心特色):怪掉結晶 → 磁吸撿取 → 背包使用注入 EXP → 升級得 3 點 → 五維能力自由分配
- 貝拉幣、死亡掉落 30% 未使用結晶、海灘重生
- 曙光嶼地形:半徑 60(主峰 + 三座副丘),高度函數同時驅動網格與碰撞;樹木/岩石程序化散布;波浪海面
- 音訊(`core/audio.ts`,WebAudio 全程式合成、零音檔):10 種動作音效(揮劍/命中/擊殺/撿結晶/撿幣/升級/跳躍/閃避/受傷/UI)+ BGM(C→Am→F→G pad 循環 + 海浪環境音),首次操作時自動啟動
- 薩爾達 3D 風格(`core/toon.ts`):cel-shading 4 階色階 + 卡通描邊 + PCFSoft 陰影;主角為綠帽劍盾勇者(程序化走路/揮劍/翻滾動畫),敵人為果凍怪(跳躍/蓄力突進/死亡爆裂)
- 手感系統(`core/fx.ts`):hit-stop 頓幀、鏡頭震動、打擊粒子、攻擊前衝、擊退、平滑鏡頭

## 和朋友一起玩(多人連線)

2~4 個家人/朋友可共玩同一座群島:看得到彼此、一起打同一批怪、各自成長,還能聊天。

- **怎麼揪人**:遊戲中按 `ESC` 開設定 →「建立多人房間」會產生一組**邀請碼**;把邀請碼(或邀請連結)傳給朋友,他在設定裡輸入邀請碼按「加入」即可進你的群島。
- **連不上就單機**:沒進房間、或伺服器睡著時,一律自動退回單機,不會卡住(斷線也會自動重連回同房間)。
- 模型:房主權威(第一個進房者當房主,離線自動移交);敵人 HP/死亡共享,寶石/結晶/任務各自入袋。詳見 `PROGRESS.md` 多人各階段。
- **自架伺服器**(可選,要讓網路上的朋友連得到才需要):`server/index.mjs` 是極簡 WebSocket 轉發站,附 `render.yaml`(Render 一鍵 Blueprint 部署)。部署後把 `wss://` 網址設成 GitHub repo 變數 `VITE_SERVER_URL`(見 `.github/workflows/deploy.yml`),前端重新部署即指向它。本機開發直接 `npm run server`(埠 8787)。
